# AIOps Đa Tác Nhân - Requirements
# 
# File này được chia thành 2 phần:
# 1. requirements-core.txt: Core dependencies để chạy hệ thống
# 2. requirements-eval.txt: Evaluation & monitoring tools (optional)
#
# Cài đặt:
# - Core system: pip install -r requirements-core.txt
# - Full system: pip install -r requirements-core.txt -r requirements-eval.txt

# Core dependencies (xem requirements-core.txt)
# LangChain, LangGraph, LLM Providers, Tenacity, Redis, OpenTelemetry, Pydantic

# Optional dependencies (xem requirements-eval.txt)
# DeepEval, Ragas, Prometheus-eval, Milvus, Pinecone, ELK Stack, Prometheus

# Để cài đặt đầy đủ:
# pip install -r requirements-core.txt -r requirements-eval.txt
